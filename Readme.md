# Yelp Review NLP Analysis
This project performs exploratory analysis on Yelp review data.

## Objective
To understand patterns in customer reviews and ratings.

## Technologies Used
Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  

## Analysis Performed
- Data exploration of Yelp reviews
- Review length analysis
- Rating distribution visualization
- Relationship between review length and ratings

## Future Improvements
- Sentiment analysis
- Text preprocessing
- Machine learning classification models
